## Getting Started

To create a new post:

1. Create a new file in `src/posts/`
2. Add frontmatter with `title`, `description`, `date`, and `tags: [post]`
3. Write your content in Markdown
4. Run `npm run build` (or let pre-commit handle it)
5. Commit and push

## Post Format

Posts support:

- **Markdown formatting**
- Code blocks with syntax highlighting
- Blockquotes
- Images
- Links

Happy writing!
