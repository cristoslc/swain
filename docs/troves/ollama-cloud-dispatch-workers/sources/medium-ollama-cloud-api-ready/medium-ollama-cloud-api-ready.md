# Ollama Cloud Inference API is now ready

By PBSEVEN, December 24, 2025

## OpenAI-Compatible Endpoints

```
https://ollama.com/v1/models
https://ollama.com/v1/chat/completions
```

## Available Models (as of late 2025 / early 2026)

cogito-2.1:671b, glm-4.6, glm-4.7, kimi-k2:1t, kimi-k2-thinking, qwen3-coder:480b, qwen3-next:80b, deepseek-v3.2, deepseek-v3.1:671b, gpt-oss:120b, nemotron-3-nano:30b, gpt-oss:20b, qwen3-vl:235b-instruct, qwen3-vl:235b, minimax-m2, minimax-m2.1, ministral-3:3b/8b/14b, mistral-large-3:675b, devstral-2:123b, devstral-small-2:24b, gemini-3-pro-preview, gemini-3-flash-preview, gemma3:4b/12b/27b.

## Authentication

Get free API key at: `https://ollama.com/settings`

Steps: Settings → API Keys → Create new key → Copy key.

## Free User Limit

Rate-limited per hour and per week. Suitable for learning and development.

## Proxy API Key Rotation

For increased limits, register multiple accounts, create API keys for each, and use a proxy key rotator to distribute requests across keys.
