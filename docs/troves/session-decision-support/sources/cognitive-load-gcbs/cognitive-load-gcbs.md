# The Impact of Cognitive Load on Decision-Making Efficiency

## Cognitive Load as a Silent Tax

- High cognitive load acts as a "silent tax on working memory," causing systematic decision quality degradation
- Working memory capacity: approximately 4 plus-or-minus 1 items (revised downward from Miller's classic 7 plus-or-minus 2)
- When total cognitive load exceeds working memory capacity, the system fails — performance does not degrade gracefully but collapses

## Effects of High Load

- Attentional narrowing: focus constricts to a subset of available information
- Inattentional blindness: relevant stimuli outside the narrowed focus are missed entirely
- Simplified information processing: complex evaluations are replaced with heuristic shortcuts
- High load shifts processing from analytical (System 2) to intuitive (System 1), increasing susceptibility to cognitive biases

## Mitigation Strategies

- Task batching: grouping similar decisions minimizes switching costs between cognitive modes
- Temporal alignment: scheduling critical decisions during biological cognitive peaks (typically morning hours)
- Standardized decision frameworks: pre-built structures reduce the per-decision cognitive cost
- Implementation intentions: "if-then" planning creates automated behavioral scripts that bypass deliberative processing

## The Role of Expertise

- Complexity is not fixed — expertise enables chunking, which reduces the cognitive cost of processing domain-specific information
- What overwhelms a novice may be manageable for an expert operating within their domain
- Chunking effectively compresses multiple items into single working memory slots
