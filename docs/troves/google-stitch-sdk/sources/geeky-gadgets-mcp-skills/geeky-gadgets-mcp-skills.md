# Build Faster with Google Stitch Using MCP, Gemini CLI & Agent Skills

**Julian Horsey** — Geeky Gadgets, January 27, 2026

## Key Takeaways

- Google Stitch introduced MCP Server, Gemini CLI Extension, and Agent Skills to streamline design-to-code workflows with AI-powered automation and integration.
- The Stitch MCP Server enables real-time AI interaction with design systems, ensuring dynamic updates, reducing repetitive tasks, and maintaining design consistency across projects.
- The Gemini CLI Extension integrates Stitch into terminal workflows, enhancing productivity with AI-driven prompts, advanced customization, and seamless command-line operations.
- Agent Skills provide task-specific automation, such as generating design documentation and converting designs into React components, with open source flexibility for customization and scalability.
- Google Stitch ensures seamless integration with popular development tools like VS Code and Cloud Code, promoting efficiency, reducing errors, and supporting collaborative workflows.

## Stitch MCP Server: Real-Time AI Integration with Design Systems

Central to these updates is the Stitch MCP Server, a cloud-based implementation of the Model Context Protocol (MCP). This server enables AI tools to interact with design systems in real time, providing deep understanding of project elements such as screens, typography, and color schemes.

The MCP Server allows AI agents to retrieve existing code or generate new designs directly within Stitch projects. This eliminates manual file transfers and ensures design-to-code workflows are accurate and efficient.

The MCP Server also supports dynamic updates, ensuring changes to design systems are reflected instantly in the coding environment, reducing inconsistencies.

## Gemini CLI Extension

The Gemini CLI Extension integrates Google Stitch into terminal-based workflows, providing developers with a seamless way to interact with the platform directly from the command line. With AI-driven design prompts that align with brand guidelines and design systems, it eliminates context switching.

Supports advanced customization options, allowing teams to tailor the extension to specific project requirements.

## Agent Skills

Agent Skills introduce specialized automation workflows to Google Stitch. Two standout skills:

- **Design MD:** Automatically generates detailed design documentation in markdown format, ensuring consistency across teams and facilitating collaboration.
- **React Components:** Converts Stitch designs into production-ready React code, adhering to best practices and maintaining design token consistency.

Built on an open source framework, Agent Skills offer developers the flexibility to customize existing workflows or contribute new ones to the community.

## Integration with Development Tools

Google Stitch provides comprehensive setup guides for integration with Cursor, VS Code, and Cloud Code. HTTP MCP server support ensures compatibility with a broad range of development tools.
