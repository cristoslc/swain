# Ollama Adds Annual Billing Option to Pro Plan at $200/year

News coverage from Efficienist about the Pro annual billing option and current cloud model lineup.

## Pro Plan Pricing

- **Monthly**: $20/month
- **Annual (new)**: $200/year ($16.67/month equivalent)
- Annual option saves the equivalent of two months vs monthly billing

## Max Plan

- **Monthly**: $100/month
- **No annual option** announced yet
- Aimed at heavier sustained usage: coding agents and batch processing

## Free Tier

Ollama's local model runner remains completely free and unlimited. Users can download and run open-source models on their own hardware without any subscription.

## Current Cloud Model Lineup

The cloud service includes access to:
- Nemotron-3-Super
- Kimi K2.5
- MiniMax M2.7
- GLM-5
- GPT-OSS

## Integration Support

Built-in support for:
- OpenClaw
- Claude Code
- Other major coding tools

## Key Distinction

The Pro and Max cloud plans are separate from local model running. Subscribers get fast, managed access to the newest open models without needing the hardware to run them locally. Local deployment costs nothing regardless of cloud tier.
