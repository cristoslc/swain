# AgentOS

## Overview

AgentOS is a mobile-first, self-hosted web UI for managing AI coding sessions across multiple runtimes. It provides multi-pane terminals, git integration, and session orchestration.

- **GitHub:** https://github.com/saadnvd1/agent-os
- **Stars:** 122
- **License:** Not specified
- **Language:** TypeScript
- **Created:** January 2026

## Runtimes supported

- Claude Code.
- Codex.
- Aider.
- Gemini CLI.
- Amp.
- Pi.

## Key features

- Mobile-first web interface.
- Multi-pane terminal management.
- Git integration and worktrees.
- Session orchestration.
- Parallel agent execution.
- Containerization support.
