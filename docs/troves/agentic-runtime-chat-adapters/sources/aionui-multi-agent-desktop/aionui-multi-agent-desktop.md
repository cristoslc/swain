# AionUi

## Overview

AionUi is a free, open-source multi-AI agent desktop application. Described as a full-model, cross-platform enhanced version of Claude Cowork. Supports multiple AI providers and CLI agents through a unified interface.

- **GitHub:** https://github.com/iOfficeAI/AionUi
- **Stars:** Not yet indexed (new project)
- **License:** Open source

## Runtimes supported

- Built-in AI agent (works immediately, sign in with Google or API key).
- Claude Code (via Multi-Agent Mode).
- Codex (via Multi-Agent Mode).
- Gemini CLI (via Multi-Agent Mode).
- OpenCode.
- Qwen Code.
- Goose CLI.
- Auggie.

## Key features

- Cross-platform desktop application.
- Remote control and monitoring for agents.
- Multi-model support: Gemini, Claude, DeepSeek, OpenAI, Ollama.
- Multi-Agent Mode for leveraging CLI agents.
- No need to install Gemini CLI or Claude Code separately (built-in agent).

## Maturity assessment

New project. Positions itself against Claude Cowork (macOS-only, Claude-locked) as a cross-platform, multi-model alternative. The breadth of supported runtimes is notable but maturity is unclear.

## Relevance to build-vs-buy

Interesting for its "cowork" model where multiple agents run through a shared interface. The cross-platform Electron approach and multi-agent mode could inform UI architecture decisions. Too early to evaluate stability.
