# Getting Started

Get started with OpenTelemetry based on your role.

Select a role to get started:

* [Dev](https://opentelemetry.io/docs/getting-started/dev/)
* [Ops](https://opentelemetry.io/docs/getting-started/ops/)

You can also try out the official [OpenTelemetry demo](https://opentelemetry.io/ecosystem/demo/) to *see* what
observability with OpenTelemetry looks like!
