# OpenTelemetry - CNCF

OpenTelemetry makes robust, portable telemetry a built-in feature of cloud-native software.

- **Organization:** open-telemetry (verified domain: opentelemetry.io)
- **Followers:** 6.9k
- **Website:** https://opentelemetry.io
- **Repositories:** 97
- **Top languages:** Go, Python, TypeScript, Java, Makefile
- **Most used topics:** opentelemetry, metrics, telemetry, distributed-tracing, tracing

## High-quality, ubiquitous, and portable telemetry to enable effective observability

OpenTelemetry is a collection of APIs, SDKs, and tools. Use it to instrument, generate, collect, and export telemetry data (metrics, logs and traces) to help you analyze your software's performance and behavior.

## Getting Involved

There is a lot to do! If you are interested in getting involved, please join the mailing lists and attend the community meetings. If you're interested in contributing to a specific part of the project, please join the appropriate special interest group (SIG). We are a friendly, collaborative group and look forward to working together!

## Security

Security-related issues should be reported through the Security Policy as security-related issues and vulnerabilities can be exploited and we request confidentiality whenever possible.

## Project Status

OpenTelemetry is a CNCF incubating project. Formed through a merger of the OpenTracing and OpenCensus projects.
