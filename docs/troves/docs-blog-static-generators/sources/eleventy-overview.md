# Eleventy (11ty) - Build Awesome

Eleventy is a simpler static site generator that recently rebranded to "Build Awesome" (March 2026).

## Recent updates

- **March 2026**: Eleventy rebranded to Build Awesome
- **May 2025**: Eleventy v3.1.0 - 11% faster and 22% smaller
- **October 2024**: Eleventy v3.0.0 stable release
- **September 2024**: Eleventy joined Font Awesome

## Key features

### Performance

- 11% faster in v3.1.0
- 22% smaller bundle size
- Zero client-side JavaScript by default

### Template languages supported

- HTML
- Markdown (with MDX support)
- JavaScript/TypeScript
- JSX
- WebC
- Nunjucks
- Liquid
- Handlebars
- Mustache
- EJS
- HAML
- Pug
- Sass

### Content features

- **Collections** - Group related content
- **Pagination** - Create pages from data
- **Data Cascade** - Flexible data inheritance
- **Permalinks** - Custom URL structures
- **Layouts** - Reusable page templates

### Plugins

- Image optimization
- RSS feeds
- Syntax highlighting
- Navigation
- Internationalization (i18n)
- Fetch remote data
- Bundle assets

## Community

- 19.5k stars on GitHub
- 17.4M downloads
- Active Discord community
- Annual 11ty Conference
- Monthly Eleventy Meetup

## Getting started

```bash
npm install -g @11ty/eleventy
```

## Why choose Eleventy

- **Simplicity** - No client-side JavaScript required
- **Flexibility** - Choose your template language
- **Performance** - Fast builds, small output
- **Stability** - Mature project with regular releases
- **Human-focused** - Community-driven development
